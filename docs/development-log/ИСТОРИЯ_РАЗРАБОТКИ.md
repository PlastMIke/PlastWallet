```
├── domain/           # Бизнес-логика
├── application/      # Бизнес-правила
```

- `WalletServiceApplication.java` - главный класс

```xml

<version>0.0.1-SNAPSHOT</version>
```

```java
}

public WalletDTO updateWallet(WalletDTO wallet) {
    return wallet;
}
```
